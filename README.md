🎓 EduAI – Multi-Agent Education System
EduAI is a Multi-Agent AI powered learning system designed to automatically generate structured educational notes using collaborative AI agents.
The system follows a modular architecture where different agents perform specific roles such as content generation, structuring, and storage.

🚀 Features
🤖 Multi-Agent Based Architecture
📝 Automatic Structured Note Generation
🗂 Topic-wise Content Storage
🧠 AI-driven Content Processing
💾 SQLite Database Integration
🌐 Streamlit-based Interactive UI
🏗️ System Architecture

EduAI follows a multi-agent workflow:
User Input Agent – Accepts topic from user
Content Generator Agent – Generates raw educational content
Structuring Agent – Formats content into clean structured notes
Storage Agent – Saves notes in database
Retrieval System – Fetches stored topics and notes

🛠️ Tech Stack
Python 3.12
Streamlit
SQLite
